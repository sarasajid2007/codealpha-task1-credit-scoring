# codealpha-task1-credit-scoring

## Overview
This task focuses on understanding and implementing the basic steps of data preprocessing and applying a simple machine learning model. The objective is to clean the dataset, prepare it for modeling, and train a basic predictive model.

## Objectives
- Import and explore dataset
- Handle missing values
- Encode categorical variables
- Normalize/scale data (if required)

- Evaluate model performance

## Workflow
1. Load dataset using Pandas
2. Perform Exploratory Data Analysis (EDA)
3. Clean data:

   - Handle missing values
4. Convert categorical data into numerical form
5. Split dataset:
   - Training set
   - Test set
6. Train model (e.g., Logistic Regression / Linear Regression / Decision Tree)
7. Evaluate model using:
   - Accuracy
   - Confusion Matrix (if classification)
   - Mean Squared Error (if regression)
